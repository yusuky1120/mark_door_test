# Todo API Application

シンプルなTodo管理アプリケーションです。  
FastAPIを用いたバックエンドAPIを中心に実装しています。

本プロジェクトはMarkdoorの面接前課題として、  
**3日間で実装可能な範囲のTodoアプリ**を作成したものです。

バックエンドの設計・実装を重視して開発しています。

## Backend
- Python
- FastAPI
- SQLAlchemy
- SQLite

## Frontend（最小構成）
- HTML
- JavaScript
- Fetch API

## その他
- Uvicorn
- Git

# 実装済み機能

- タスク作成
- タスク一覧取得
- タスク更新
- タスク削除
- タスクの完了 / 未完了切り替え
- タスクフィルタリング（完了 / 未完了）
